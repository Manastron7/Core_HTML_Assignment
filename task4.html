Tags information

  1. HTML
    • The html tag is the root element of an HTML document. It encloses the entire webpage's content and defines it as an HTML document.

  2. HEAD
    • The head tag contains metadata about the webpage, such as the title, character set, links to stylesheets, and scripts. It doesn't display any visible content on the page

  3. a tag
    • The a tag is used to create hyperlinks or anchor links. It allows you to link to other web pages, resources, or even sections within the same page.

  4. SMALL
    • The small tag is used to indicate that the text within it should be displayed in a smaller font size than the surrounding text. It is typically used for fine print, disclaimers, copyright notices, or any content that should be visually smaller but still legible. It's a presentational element and does not convey any semantic meaning on its own.

  5. h1 to h6
    • These tags are used to create headings with different levels of importance. h1 is the highest level (most important), while h6 is the lowest.

  6. SUB
    • The sub tag tag is used to format text as subscript, meaning it appears slightly below the baseline of the surrounding text. Subscript text is commonly used for mathematical notations example H2O for water or chemical formulas. It can also be used for footnotes or references.
  7. p tag
    • The p tag is used to define paragraphs of text. It creates a block-level element with space before and after, making it suitable for long-form textual content.
