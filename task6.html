  2. CSS

    3. Primary use Cases Key Features/ Benifits
    CSS used to style and design web pages
    • It separates the presentation from the content
    • It works independently of HTML
    • It ensures a consistent design across webpages
    • It reduces redundancy in styling
    • It allows websites to adapt to different devices screens

  3. JS

    4. Primary use Cases Key Features/ Benifits
    JavaScript add interactivity and dynamic behaviour to websites

    • It doesn't require constant server communication
    • It has a vast ecosystem and numerous libraries
    • It supports asynchronous operations
    • It is used as a single language to build full-stack applications
